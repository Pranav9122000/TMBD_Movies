# 🎬 TMDB Movies

A movie discovery iOS application built with SwiftUI using Clean Architecture.

## Features

- Browse trending and popular movies
- Movie details
- Async/Await networking
- MVVM + Clean Architecture
- Repository Pattern
- Dependency Injection

## Tech Stack

- Swift
- SwiftUI
- Async/Await
- URLSession
- MVVM
- Clean Architecture

## Architecture

This project follows **Clean Architecture** with MVVM, Repository Pattern, and Dependency Injection.

For a detailed explanation of the project architecture, see **[Architecture Documentation](./architecture_plan.md)**.
